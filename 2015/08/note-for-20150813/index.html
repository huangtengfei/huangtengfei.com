纯 CSS 制作 Tab 选项卡

有两种使用纯 CSS 制作 Tab 选项卡的方式。

第一种：使用使用表单的单选按钮（radio）来实现当前 Tab 显示和隐藏，首页 Tab 里的内容默认隐藏，如果单选按钮为选中状态（:checked）就显示内容。

参考 纯CSS3制作的Tab选项卡 可以快速实现一个demo。

第二种：通过点击 :target 时令对应的z-index变大，重复覆盖在上一层。可参考 CSS3-Only Tabbed Area 。不过并不推荐。

参考：纯CSS3制作TAB选项卡的两种方式 和 一个很漂亮的使用第一种方式实现的 demo

CSS outline 属性

outline （轮廓）是绘制于元素周围的一条线，位于边框边缘的外围，可起到突出元素的作用。

可以按顺序设置如下属性：

  • outline-color
  • outline-style
  • outline-width

比如在写搜索框的时候，可能不想要 input focus 时默认的 outline，而是要一个发光的效果，可以这样实现：

为 tr 添加底部 border

在设置 table 的 tr 的 border-bottom 时，不生效。

解决办法：

将该 tr 所在 table 的 css 属性 border-collapse 设置为 collapse 。

border-collapse 用来设置表格的行和单元格的边框是合并在一起还是按照标准的HTML样式分开。

可用值的说明：

  • separate 缺省值。边框分开，不合并。
  • collapse 边框合并。即如果相邻，则共用同一个边框。
  • inherit 从父元素继承 border-collapse 属性的值。

参考：CSS outline 属性 和 css3 - input focus的发光效果
