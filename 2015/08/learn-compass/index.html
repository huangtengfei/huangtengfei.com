Compass 初探

发表于 2015-08-04   |  

Sass 本身只是一个编译器，Compass 在它的基础上，封装了一系列有用的模块和模板，补充 Sass 的功能。它们之间的关系类似于 Javascript 和 jQuery。

安装

在已有Ruby的环境下，在命令行输入命令：

gem install compass

使用

compass create

使用 compass create 命令可创建一个 Compass 项目：

compass create myproject

该命令会生成以下目录结构：

  • sass目录，存放 sass 源文件
  • stylesheets目录，存放编译好的 CSS
  • config.rb文件，Compass 的配置文件

在 config.rb 文件中，output_style = :expanded 可以指定编译后的 CSS 格式。也可以通过指定 environment 的值（:production或者 :development），智能判断编译模式。

compass compile

执行 compass compile 命令可以根据 config.rb 中设置的条件，对 Sass 源文件编译。

compass watch

compass watch 命令可以对 config.rb 中指定的目录进行监控，自动编译。

Compass 模块

Compass Core内置六个模块，分别为

  • Reset
  • CSS3
  • Helpers
  • Layout
  • Typography
  • Utilities

这里介绍常用的 Reset 模块、CSS3 模块、Helpers 模块和 Utilities 模块。

Reset 模块

通常，编写自己的样式之前，有必要重置浏览器的默认样式。

使用 @import 命令，加载 reset 模块，编译后，会生成相应的 css reset 代码。

@import "compass/reset";
