访问兄弟节点

由于IE以外的其他浏览器不会忽略节点间生成的空白文本节点（例如换行符号），因此 nextSibling 属性返回的某个节点之后紧跟的节点可能并不符合预期（处于同一树层级中）。

解决办法：判断节点 nodeType 是否为1, 如是为元素节点，跳过。

几个位置概念

scrollLeft 与 scrollTop

document.body.scrollLeft：网页被卷去的左，即对象最顶端与窗口中可见内容的最顶端之间的距离。

document.body.scrollTop：网页被卷去的高，即给定对象左边界与窗口中目前可见内容的最左端之间的距离。

clientX 与 clientY

鼠标事件对象 MouseEvent 下的属性。

以可视区域左上角为参考点，表示当前触发点离可视区域左上角的水平和垂直距离。以当前浏览器为参照物，不受滚动条影响。

offsetLeft 与 offsetTop

元素位置的属性（固定，不受鼠标事件的触发点影响）。

元素 border 外边框的左上角离 offsetParent（布局中设置postion属性(Relative、Absolute、fixed)的父容器）的 padding 外边框的左上角的垂直、水平距离。

更多可参考 JS魔法堂：关于元素位置和鼠标位置的属性

关于滚动翻页用到的几个位置请参考 学习笔记 2015.08.24

history对象

history对象记录了用户曾经浏览过的页面(URL)，并可以实现浏览器前进与后退相似导航的功能。

window.history.[属性|方法]

window.history.back()：前一个url，相当于go(-1)

window.history.forward()：下一个url，相当于go(1)

window.history.go()：跳到某个具体的url

location 对象

location用于获取或设置窗体的URL，并且可以用于解析URL。

location.[属性|方法]

location.href：返回完整url

location.pathname：路径名

location.hash：返回#开始的url锚

navigator 对象

navigator.appName：浏览器名称

navigator.appVersion：浏览器版本
