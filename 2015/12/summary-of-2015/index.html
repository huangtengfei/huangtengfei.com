2015 小结

发表于 2015-12-28   |  

时间过得真快，马上 2015 就要完了，作为毕业后工作第一年，好像是该写点什么。

去年 12 月底，在忙完校招答辩等一大堆事情后，觉得闲着也是闲着，就跟人事商量了以下提前过来实习。很幸运，有飞哥 @徐飞 作为带教，提前联系了一下。12 月 22 号，来南京开始了实习。

实习时间持续两个多月，在此期间，做了一个类似 LeanCloud 的 BaaS demo，并对开发中遇到的问题做了一个 总结 ，写了 使用说明 。只不过当时没怎么用 github，代码全在本地。另外，使用 hexo 基于 Github Pages 搭建了一个 个人博客 ，对 Git基本命令 、前端工具 grunt、bower 等 和 JavaScript 基本语法 做了简单学习。

在实习期间，跟一个喜欢健身的小伙伴去健身了有一个多月时间，本想后面一直坚持着，说不定能长点肉，可惜后来他不在这边工作了，自己便也没去过了。

3 月中旬结束了实习，然后回学校领了毕业证学位证，回家休养了一下，4 月 1 号正式入职，并在四月中旬参见了半个月变态的入职集训（好在认识了一帮小伙伴），一晃到了 5 月。

5 月和 6 月的时间基本上都花在了云商城一期项目上，算是把 Angular 用的更熟练一点，然而因为一开始没参与（接手了别人代码），而且当时没看过代码规范，导致代码写得很烂，现在都不想看之前写的代码。。。

在这两个月，还一个很坑的事情是染指了股票，导致被经济问题困扰了好久，不过也并不后悔，算是交了 学费 吧。

到了 7 月份，项目一期结束，算是相对轻松了一些，断断续续学习了一些 Web 开发的最佳实践，比如 CSS 最佳实践，还有 正则表达式，jQuery 和 Sass，也看了一些 设计模式 的书籍，不过还是觉得这种东西需要从实际项目中归纳总结，不然没什么感觉。

从 8 月份开始，在二期项目之余，通过 百度技术学院 ，先后学习了 HTML 和 CSS 基础，对 布局、浮动 等有了更深入的理解，并写了一个 简单博客 练手。学习了 JavaScript 基础，对其中 作用域、原型、闭包、面向对象、this 有了更深入的理解，并写了一个简单的 JavaScript 工具库，也实现了一些轮播、拖拽、输入提示等效果。学习了 HTML 编码规范、CSS 编码规范、JavaScript 编码规范、JSON 数据传输规范 和 项目目录结构规范，最后实现了一个 简单的 个人任务管理系统 （本来是应该写完推倒重来的，后来新项目开始忙就没重构了。。。）。整个过程大概持续了一个半月。

8 月份尝试了一段时间的跑步，无奈对体重的伤害太大，最终也没坚持下来。两次健身计划都中止，大概是今年最让我遗憾的事情了，也间接导致了11、12 月份身体状况一直不佳。

9 月中旬开始，参与了一个月左右 持续交付平台项目 的开发，通过搭前端架子，对 Angular 各模块理解更深了一点，根据项目需要，也学了一下 ECharts 的使用，代码质量比前一个项目稍微好了一点。

9 月底到 10 月初，在项目之余对 Angular 中 Controller 之间的通信、作用域、数据绑定、各种 Provider 的用法、ui.router 的使用 和 常用模块的书写规范 做了一个学习和总结，并建了一个 angular-demo 库来进行汇总，也对 ES6 做了一个 入门学习，这些都在后面项目的开发和自己写的 demo 中得到了应用。

10 月中旬往后的一个多月时间，除了日常的项目开发，主要时间都花在了 前面提到的 BaSS demo 的重构上（其实几乎是用 ES6 重写了），本打算在 2015 结束之前发布一个版本，没想到后面项目太忙，只能又搁置了。不过基本的存储功能已实现，也写了一个新的 使用说明，后期再继续完善。期间帮豆芽那边实现了一个需求，了解了一下 zTree 并写了一个 demo，把八月份挖的一个 拯救单身狗 的坑填完，简单学习了 webpack 的使用并写了 demo（还差一篇教程要写），学习了一下 基于 HTML5 的多图片预览上传，并写了包括前端库和后台逻辑的完整 demo。

12 月接了新活，开始了私有云那边的开发，因为前面的学习，而且加上这次个人负责的模块完全从零开始写，所以把各种书写规范和 ES5 新特性用上了，写得代码也明显比之前项目高很多，看着也让人神清气爽。。。而且嫌模拟数据费事，把之前自己做的 BaaS demo 的后台提取出来一部分，做了一个 mock-server，用 Node.js 和 MongoDB 来提供后台逻辑和存储，用起来简直太爽。

此外，在飞哥带领下全组小伙伴去参加了 D2 技术论坛，收获满满，此前担心 D2 上讲 React 自己听不懂，还抽时间入门了下 React，了解了一下 Virtual DOM，写了一个 demo。

整个一年差不多就做了这些事情，总体来说收获还是蛮多的，对前端也越来越喜欢。而且，最重要的，认识了飞哥这样的大神和一帮很欢乐的小伙伴，每天的工作都是开心的。。。（虽然偶尔也会因为某些原因郁闷一下）

流水账完了，下面开始立 flag 了，2016年，大致的计划有以下这些。

身体方面：

  1. 坚持健身，不希望明年冬天的身体再像今年这么差（考验毅力的时候到了）

  2. 长胖十斤（考验大厨水平的时候到了）

学习方面：

  1. 深入学习一下 JavaScript（红宝书、深浅 Node.js） 和 CSS（DOM，CSS3），基础很重要

  2. 每周看一定数量技术文章，并坚持每周至少一篇博客（注重深度）

  3. 至于框架，Angular2 和 React 这些，根据需要来学习

项目方面（除了日常开发项目）：

  1. 将前面未完成的 BaaS demo 完成，提供基本的存储和实时推送功能，部署在公司服务器上提供服务

  2. 实现一个 Web 接口管理工具，提供可配置的接口，用来做 Mock Server （推广使用）

  3. 开发一个 APP（已有两个想法，做哪个待定），并在 APP Store 上线

日常：

知乎多答题，目标 500 关注。

最后，希望明年能有比今年更好的学习心态。悟已往之不谏，知来者之可追。

坚持原创技术分享，您的支持将鼓励我继续创作！
