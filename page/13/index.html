CSS 定位

发表于 2015-07-01   |  

display属性

display 属性规定元素应该生成的框的类型。一切皆为框，比如块级元素div、h1、p生成的“块框”，内联元素strong、span生成的“行内框”。

position属性

CSS 有三种基本的定位机制：普通流（static、relative）、浮动（float）和绝对定位（absolute、fixed）。

正常情况下，所有框都在普通流中定位，块级框从上到下一个接一个地排列（框之间的垂直距离由框的垂直外边距计算出来），行内框在一行中水平布置（可以使用水平内边距、边框和外边距调整它们的间距）。
