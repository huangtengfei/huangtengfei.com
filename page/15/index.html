基于Start快速构建一个Web应用

发表于 2015-02-05   |  

Start 是一个 BaaS 产品的雏形，它提供了基本的服务器端 RESTful 风格的 API （包括 HTTP API 和 Socket API） 、基于 MongoDB 的数据库存储和实时推送功能。基于 Start ，你可以快速构建一个 Web 应用，你需要做的工作只是完成Web应用的前端开发和数据库（表）的设置。

Start 使用以下技术和库进行开发:

  • Node.js
  • Express
  • Mongoose
  • AngularJS
  • Bootstrap 3
  • Socket.io

创建数据库

首先，你需要创建一个数据库，用于为你的应用提供数据存储服务。点击 New DB 按钮，在数据库创建页面，输入数据库名称，然后点击 Create DB 按钮，数据库即创建完成。

数据库创建完成后，会跳转到 DB List 页面，点击 DB 后面的文件图标，即可查看该 DB 的 APP ID，此 ID 用于配置 Web 应用的数据库。
